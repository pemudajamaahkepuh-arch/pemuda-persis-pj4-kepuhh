# pemuda-persis-pj4-kepuhh
Pemuda Persis Kepuh PJ IV Kepuh merupakan wadah pembinaan generasi muda yang berlandaskan nilai-nilai Islam sesuai dengan manhaj Persatuan Islam (Persis). Organisasi ini berperan aktif dalam membentuk karakter pemuda yang berakhlak mulia, berilmu, disiplin, serta memiliki kepedulian terhadap masyarakat dan lingkungan sekitar.
